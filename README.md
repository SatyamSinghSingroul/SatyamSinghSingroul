## 📊 Featured Projects

### 🔄 [Telecom Customer Churn Analysis](https://github.com/SatyamSinghSingroul/Telecom-Customer-Churn-Analysis)
> Python · SQL Server · Power BI · Random Forest

End-to-end analytics pipeline on **6,418 telecom customer records** — from SQL Server ETL to churn prediction and Power BI dashboard.

- Built ETL pipeline in SQL Server: staging, null handling, production views
- Identified month-to-month contracts **(46.5% churn)** and fibre optic internet **(41.1%)** as highest-risk segments
- Random Forest model achieved **85% accuracy** · Predicted **370 future churners** from new joiners
- Two-page Power BI dashboard with churn KPIs, geographic breakdown, and prediction page
